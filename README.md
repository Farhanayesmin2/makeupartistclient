## Install Tailwind CSS

For tailwind install follow this step,
1.npm install -D tailwindcss
npx tailwindcss init

2.Add the paths to all of your template files in your tailwind.config.js file.
content: ["./src/**/*.{html,js}"],

3. Add the @tailwind directives for each of Tailwindâ€™s layers to your main CSS file, add index.css
   @tailwind base;
   @tailwind components;
   @tailwind utilities;

4.Add link on index.html <link href="/dist/output.css" rel="stylesheet"> and again npm start
